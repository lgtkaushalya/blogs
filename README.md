blogs
=====

A Symfony project for a workshop at University of Moratuwa

Installation Instructions
========================

1. Get a git clone or download the source.
2. Create your database as "blogs" and source the blogs.sql script
3. Run "composer install" to install dependancies.
4. Change the "app/config/parameters.yml.dist" as "app/config/parameters.yml" and modify the parameters. Usually this step will be completed through a interactive window in above step.
5. Give write permissions for web/ cache/ and log/ folders.
6. Access blog list via "<hosted url>/web/app.php/blogs" url.
